# GrayHare.Snake

## Overview

GrayHare.Snake is a grid-based Snake game built with [`GrayHare.GameEngine`](https://github.com/DdeHaas/GrayHare.GameEngine). Guide the snake around the board, eat food to grow, collect bonus food for extra points, and avoid walls and your own body while pushing for a higher score.

## Controls

| Action | Keyboard | Gamepad |
|---|---|---|
| Start game | Any key | Button 0 |
| Move up | `Up`, `W` | D-pad / POV Y |
| Move down | `Down`, `S` | D-pad / POV Y |
| Move left | `Left`, `A` | D-pad / POV X |
| Move right | `Right`, `D` | D-pad / POV X |
| Pause | `P` | Button 7 |
| Exit from title screen | `Escape` | — |

## Running from the repository

```
dotnet run GrayHare.Snake
```
